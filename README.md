## Descripción

Proyecto de titulación de una plataforma de participación ciudadana.
Esta es la base de un proyecto realizado con el framework NestJS.

## Instalación

```bash
$ npm install
```

## Ejecutando la aplicación

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
